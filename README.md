# LvMP
- - - 
## 1. 설명
로컬 음악재생 iOS앱
- - -
## 2. 기능
아이튠즈를 통해 폴더 구조로 음악을 넣을 수 있고, 해당 폴더 구조를 유지한체 음악목록을 볼 수 있다.
- - - 
## 3. 화면 구성
메인 화면: 최근 추가된 앨범 리스트, 노래 리스트
아티스트: 아티스트(가수) 목록 -> 해당 아티스트의 앨범 및 하위 노래 목록
앨범: 앨범 목록 -> 하위 노래 목록
폴더: 폴더 목록
